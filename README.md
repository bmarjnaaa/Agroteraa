## üõ†Ô∏è Getting Started

### üìã Prerequisites

This project requires the following dependencies:

- **Programming Language:** PHP
- **Package Manager:** Composer, Npm

### üíæ Installation

Build Agrotera from the source and install dependencies:

1. Clone the repository:

    ```bash
    git clone https://github.com/jaaaalagi/Agrotera
    ```

2. Navigate to the project directory:

    ```bash
    cd Agrotera
    ```

3. Install the dependencies:

    Using **composer**:

    ```bash
    composer install
    ```

    Using **npm**:

    ```bash
    npm install
    ```

4. Copy environment configuration file and configure variables:

    ```bash
    cp .env.example .env
    ```

    Edit `.env` to set up your database connection, app URL, and other environment variables.

5. Generate application key (for Laravel projects):

    ```bash
    php artisan key:generate
    ```

6. Run migrations and seed the database (if applicable):

    ```bash
    php artisan migrate --seed
    ```
